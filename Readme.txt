Income Expense Calculator

This project is a simple Income and Expense calculator that allows users to add, edit, delete, and filter income and expense entries.
 It provides a financial overview showing the total income, total expenses, and the net balance.

Features
- Add new income and expense entries.
- Edit existing entries.
- Delete entries from the list.
- Filter entries by type (income, expense, all).
- Displays total income, total expenses, and net balance
